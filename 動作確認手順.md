## 🐛 トラブルシューティング

### 問題A: 都道府県が表示されない

**症状：** プルダウンに「選択」しか表示されない

**確認方法：**
1. F12キーを押して開発者ツールを開く
2. Consoleタブを確認
3. 赤いエラーメッセージがあるか確認

**よくあるエラー：**
- `Papa is not defined` → PapaParse読み込み失敗
- `Failed to fetch` → CSV URL アクセス失敗
- `CORS error` → CSVへのアクセス権限問題

**対処：**
エラーメッセージをコピーして報告してください。

---

### 問題B: CSVデータが読み込めない

**症状：** エラーメッセージ「データの読み込みに失敗しました」が表示される

**確認方法：**
1. CSV URLに直接アクセス：
   ```
   https://docs.google.com/spreadsheets/d/e/2PACX-1vSkGeGpXlosW_k2NKZBuhnhI_y3qex4Oi8HZRx5IyFC8RBRKWpwy7C3ZeXgWtjnQw5Vc-iDU-W6YeUa/pub?gid=0&single=true&output=csv
   ```
2. CSVデータがダウンロード/表示されるか確認

**対処：**
- CSVが表示される → ブラウザのCORS問題の可能性
- CSVが表示されない → Googleスプレッドシートの公開設定を確認

---

### 問題C: デザインが崩れている

**症状：** フォントが正しくない、色が表示されない

**確認方法：**
1. F12 → Networkタブ
2. Google Fontsが読み込まれているか確認
3. CSSが適用されているか確認

**対処：**
- ハードリロード（Ctrl + F5）を試す
- ブラウザキャッシュをクリア

---

## 📊 開発者ツールでの詳細確認

### CSV読み込み状況を確認する方法

1. F12で開発者ツールを開く
2. Consoleタブに以下を入力して実行：

```javascript
// データが読み込まれているか確認
console.log('都道府県数:', appData.rowsByPref.size);
console.log('プロバイダー:', appData.providers);
console.log('カテゴリ:', appData.categories);
```

**期待される結果：**
```
都道府県数: 47
プロバイダー: ['工務店', '工務店', '工務店', '工務店', '職人', '職人']
カテゴリ: ['注文住宅', '大規模', '小規模', 'オフィス・店舗', '小規模', 'オフィス・店舗']
```
