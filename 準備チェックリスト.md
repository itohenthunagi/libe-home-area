# 実装前準備チェックリスト

## ✅ 必須の準備項目

### 1. Googleスプレッドシートの準備

#### 1.1 公開用シート「公開_まとめ」の作成

**手順：**
1. 既存のスプレッドシートを開く
2. 新しいシートを追加して「公開_まとめ」と名前を付ける
3. 以下の構造でデータを配置する：

```
行番号 | A列(都道府県) | B列(市区町村) | C列(ふりがな) | D列以降(判定セル)
-------|--------------|--------------|--------------|------------------
1      | (任意・未使用でもOK)
2      | (空白)       | (空白)       | (空白)       | プロバイダー種別（工務店、職人など）
3      | (空白)       | (空白)       | (空白)       | カテゴリ（大規模、小規模など）
4以降  | 都道府県名    | 市区町村名    | ふりがな      | 判定値（○、要相談など）
```

**例：**
```
1 | # 全エリアの対応状況
2 |              |              |              | 工務店    | 工務店  | 職人
3 | 都道府県      | 市区町村      | ふりがな      | 大規模    | 小規模  | 小規模
4 | 北海道       | 札幌市        | さっぽろし    | ○        | ○      |
5 | 北海道       | 函館市        | はこだてし    |          |        |
```

#### 1.2 データ転記（非公開シートから公開用シートへ）

**方法A：全列コピー（簡単だが注意が必要）**
- `公開_まとめ` のA1セルに：`=ARRAYFORMULA(まとめ!A:Z)`
- ⚠️ 非公開情報が含まれないか要確認

**方法B：必要列のみ転記（推奨）**
- `公開_まとめ` のA1セルに：`=ARRAYFORMULA(まとめ!A:V)`
  （必要な列まで範囲を指定）
- または列を個別に指定：`={まとめ!A:C, まとめ!D:V}`

**確認ポイント：**
- [ ] 都道府県・市区町村・ふりがなが正しく入っているか
- [ ] ヘッダ行（2行目・3行目）が正しく設定されているか
- [ ] 判定セル（D列以降）に想定通りの値が入っているか
- [ ] **非公開情報（個人情報、社内メモなど）が含まれていないか**

### 2. CSV公開URLの取得

**手順：**
1. Googleスプレッドシートを開く
2. メニューバー → **ファイル** → **共有** → **ウェブに公開**
3. 「公開するアイテム」で **「公開_まとめ」シート** を選択
4. 形式：**「カンマ区切り形式（.csv）」** を選択
5. **「公開」** ボタンをクリック
6. 表示されたURLをコピー

**URLの形式：**
```
https://docs.google.com/spreadsheets/d/【スプレッドシートID】/export?format=csv&gid=【シートID】
```

**確認方法：**
- コピーしたURLをブラウザで開く → CSVデータがダウンロードまたは表示される
- 内容が「公開_まとめ」シートと一致しているか確認

**✍️ ここに取得したURLを記録してください：**
```
CSV_URL =

```
